This folder contains preprocessed MR data for each participant, mainly including nifti data after HeuDiConv and derivatives of fMRIPrep.

## Files Structure:
- **derivatives**, including preprocessed MR data derived from fMRIPrep
- **sub-##**, including converted nifti data derived from HeuDiConv
- **HeuDiConv processing details**, including participants.csv, dataset_description.json, etc.
